# Card World — System Architecture

- **Everything Is Card** — fixed Title / Image / Text, ratio 5:7.
- **Computer** = container card + `scene_push`, not a JS class.
- **IR + VM** — same semantics in-world and in engine.
- **No meta economy / power / approval** early on.
- **Packages:** slug, title, version, author; Install (free), Publish, Source.
- **Phases:** 0 demo → 1 Computer/Store/flagship → 2 other players Install → 3+ optional tip.
- **Networking:** list/install/publish/save only; single-player feel.

See **DEMO.md** for playbook and tools.
