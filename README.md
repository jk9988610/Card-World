# Card World

Everything is a card. Program the world from Hand + Field.

## Play online

**https://jk9988610.github.io/Card-World/**

Repo **Settings → Pages → Build and deployment** must use **GitHub Actions**.

## Local

```bash
npm run build-seed
npx --yes serve . -p 3000
```

Open http://localhost:3000
