# Offline instrument samples

## INS-008 (piano)

`INS-008/` contains Salamander grand piano multisamples (from [Tone.js audio assets](https://github.com/Tonejs/audio/tree/master/salamander), bundled for offline use).

Paths are relative to `embedded/harmonyforge/index.html` (`samples/INS-008/*.mp3`).

To add a custom **sampler** slot, create `samples/<INS-id>/` and set `kind: "sampler"` in `js/instrument-registry.js`.
